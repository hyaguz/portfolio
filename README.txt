Portfólio - Carros
Gerado para Hyago.

Arquivos:
- index.html : página principal
- assets/style.css : estilos
- assets/script.js : interatividade (modal + scroll)
- assets/instagram.svg : ícone do instagram usado no botão

Imagens:
As imagens são carregadas via Source Unsplash (dinâmico). Três imagens usam 1600x900 (ultra) e duas usam 800x600 (otimizadas).

Como usar:
1. Descompacte portfolio_carros.zip
2. Abra index.html no navegador
